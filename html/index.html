About Shutdown Timer

Shutdown timer is an application to shutdown your linux based home computer by a timer. It uses "halt" command for shutingdown your computer.

So to use this you must run this as root. When it ask password type your password.

CAUTION :

Do not use this on Servers, Public computers, Office, or even on your fathers computer.

Install

To install first extract the files, then type "sudo ./setup_shtimer" then it will do the magick for you.

The UI

The power of QT is so good. This uses Qt as the basic.

The Timer

Use the combo boxes to set the time.

The Control

Use pushbuttons to Start and Cancel the timer.

The Display

This shows the remaining time for your shutdown.
